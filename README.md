# CalculatorApp

- Course name: Introduction to Mobile Systems
- Lab number: Lab Task 3
- Student name: Cem Cakir

CalculatorApp is an Android Studio project written in Java. The app implements a calculator with basic arithmetic operations, base conversion, history, and an expression calculator screen.

## Build

Open the project in Android Studio or build with:

```bash
./gradlew assembleDebug
```

The generated debug APK is located at `app/build/outputs/apk/debug/app-debug.apk`.
